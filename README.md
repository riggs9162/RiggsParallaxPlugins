# Riggs' Parallax Plugins

This repository serves as a collection of plugins designed for the Parallax Framework within Garry's Mod. These plugins enhance gameplay and provide additional functionality to servers running the framework.

## Installation

1. Download or clone this repository.
2. Place the plugin folders into your Garry's Mod server's `garrysmod/gamemodes/*yourschema*/modules/` directory.
3. Ensure the Parallax Framework is installed and configured on your server.
4. Restart your server or reload the plugins as needed.

## Usage

Once installed, the plugins will integrate with the Parallax Framework. Refer to the framework's documentation for detailed configuration options and usage instructions.
